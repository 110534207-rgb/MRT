# 專案概觀

本專案為一個簡單的靜態 Web 應用程式，用於顯示台北捷運（大眾捷運系統）的路線圖。它使用 HTML、CSS 和原生 JavaScript 建置。

該應用程式為不同的捷運路線提供互動式地圖。使用者可以點擊車站以查看附近的景點列表。還有一個受密碼保護的簡單管理員模式，可以為每個車站新增和刪除景點。

捷運車站和景點的資料直接儲存在每個 HTML 檔案的 JavaScript 程式碼中。專案的結構是，主目錄包含每個捷運路線的範本檔案，而子目錄 `110534207` 則包含這些檔案的客製化版本。

# 建置與執行

這是一個靜態 Web 專案，沒有建置過程。若要執行此應用程式，只需在網頁瀏覽器中開啟任何 `.html` 檔案即可。

例如，您可以開啟 `捷運板南線.html` 來檢視板南線。

# 開發慣例

*   **框架/函式庫：** 本專案使用原生 HTML、CSS 和 JavaScript，沒有任何外部框架或函式庫。
*   **樣式：** CSS 直接包含在每個 HTML 檔案的 `<style>` 標籤中。
*   **資料：** 車站和景點資料儲存在每個 HTML 檔案中 `getDefaultStations` 函式內的 JavaScript 陣列中。
*   **管理員存取：** 點擊「管理員登入」按鈕並輸入密碼 `123` 即可進入管理員模式。
*   **檔案結構：** 每條捷運線都有其獨立的 HTML 檔案。`110534207` 目錄似乎是使用者或學生特定的目錄，其中包含他們的作業。
